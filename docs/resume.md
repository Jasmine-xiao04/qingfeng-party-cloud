# 简历项目描述

## 产品岗版本

青锋党建云｜高校党支部活动与积分管理平台  
基于党支部学生助理实际工作场景，独立完成从需求调研、PRD 设计、信息架构、权限模型、MVP 范围拆解到可运行系统落地。项目解决活动名单人工汇总、积分手动统计、排名更新滞后、活动通知不清晰等问题，设计了 Excel 导入预览、积分流水追溯、同批次排行榜、必参加活动提醒和荣誉展示模块。通过流程数字化，将活动名单统计和积分核算从手工表格维护转为系统自动处理，显著降低重复劳动和统计错误。

## 前端/全栈版本

青锋党建云｜React + Express + Prisma 全栈项目  
使用 React、TypeScript、Vite、Ant Design、Express、Prisma 和 PostgreSQL 开发高校党支部管理系统，实现 JWT 登录认证、三类角色权限、成员管理、活动发布、Excel 活动名单导入、历史积分导入、积分流水、排行榜、学生端首页和荣誉展示。核心导入流程支持 Excel 解析、学号优先匹配、同名冲突提示、导入预览确认和积分流水自动生成，保证数据可追溯。项目采用前后端分离与响应式布局，可部署到 Vercel、Render/Railway 和 Supabase。

## 项目亮点

- 真实校园党建场景，问题来源明确。
- Excel 批量导入替代人工统计，降低支部助理重复劳动。
- 积分流水机制保证数据透明、可追溯。
- 管理端与学生端分离，角色权限清晰。
- 支持 PC 和手机浏览器访问。
- 具备 README、数据库设计、接口文档和部署文档，适合 GitHub 展示。
