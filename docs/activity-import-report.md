## 页面化导入能力

当前系统已支持在管理端页面完成活动名单导入：

```text
管理端 -> 名单导入 -> 选择活动 -> 选择 Excel -> 生成导入预览 -> 确认入库
```

支持识别 `姓名（必填）`、`学号（必填）` 这类问卷收集表表头。预览阶段会提示未匹配、重复记录、同一活动重复导入等问题。
